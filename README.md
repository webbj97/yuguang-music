<h1 align=center>网易云音乐（余光）</h1>


### 运行步骤

```
clone 仓库地址
cd 代码仓库
yarn install || npm install
yarn dev || npm run dev
```

### 技术栈

* 语言：Vue、Sass
* 接口：axios
* 状态管理：Vuex
* 组件：element
